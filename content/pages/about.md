# About This Site

Welcome to the about page! This site demonstrates the power of Pelican, a Python-based static site generator.

## What is Pelican?

Pelican is a static site generator that allows you to:

- Write content in Markdown or reStructuredText
- Use Jinja2 templates for theming
- Generate a complete static website that can be hosted anywhere
- Version control your entire site with Git
