About web2site

web2site is a demonstration project showcasing the power and elegance of static site generation using modern web technologies.

Our Mission

We believe in the power of simplicity. In a world of complex content management systems and heavy frameworks, static sites offer a refreshing alternative that prioritizes:

  • Performance: Deliver content at lightning speed
  • Security: Minimize vulnerabilities through simplicity
  • Maintainability: Keep codebases clean and understandable
  • Accessibility: Ensure content is available to everyone

Technology Stack

This site is built with carefully chosen technologies:

Pelican Static Site Generator

Pelican is a Python-based static site generator that transforms Markdown files into beautiful HTML pages. Key features include:

  • Markdown Support: Write content in simple, readable Markdown
  • Jinja2 Templating: Powerful template engine for customization
  • Plugin System: Extend functionality with community plugins
  • Multi-format Output: Generate RSS/Atom feeds automatically

Octopress Theme

The Octopress theme provides a clean, minimalist design that emphasizes content readability and user experience. Originally created for the Octopress blogging framework, it's been adapted for Pelican to bring its elegant design to a wider audience.

GitHub Pages Hosting

Deployed automatically via GitHub Actions, ensuring:

  • Zero hosting costs
  • Automatic deployments on every commit
  • Global CDN distribution
  • HTTPS by default
